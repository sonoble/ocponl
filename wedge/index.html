Open Network Linux - Getting Started With The Wedge

  • ONL Supports the [Wedge 40] running the Facebook [FBOSS] fowarding agent and [Snaproute].
  • ONL Supports the Wedge 100 running [Snaproute].

Installation Instructions

  1. If your Wedge does not have ONIE, you will need to install it to install ONL, it can be download from here:

    1. Wedge 40 - ONIE for Wedge 40
    2. Wedge 100 - ONIE for Wedge 100

  2. Burn the ONIE .iso to a USB

  3. Assuming that your USB is /dev/sdb, under linux you would do: dd if=onie-recovery-x86_64-accton_wedge_x16-r0.iso of=/dev/sdb bs=10M

  4. Follow the directions in Wedge Installation Guide
